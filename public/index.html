Every branch is a live website.

This project is an experiment in autonomous deployment. Claude Code sessions create branches, build solutions, and push them to this repository. Each push triggers a GitHub Action that deploys the branch to its own unique URL on Cloudflare's global edge network.

The production site — the page you are reading now — lives on main. Every other branch gets a preview deployment at its own subdomain. All of them are protected behind Cloudflare Access, so only authorized viewers can see the work in progress.

There is no build step. No framework. No configuration beyond what ships in the repository. Push code, get a URL. That's the entire workflow.

How It Works

1. A Claude Code session creates a branch. It writes a solution — a page, a tool, an experiment — and pushes the branch to GitHub.

2. GitHub Actions deploys it. The push triggers a workflow that deploys the public/ directory to Cloudflare Pages. The branch name becomes the subdomain: feat-dashboard.solutions-hub-1rh.pages.dev.

3. Review and merge. Visit the preview URL, review the solution. If it's good, merge to main. Production updates within seconds. The preview URL is deactivated when the branch is deleted.
