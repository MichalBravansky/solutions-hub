How It Works

Push to Deploy

Three steps. That's it.

01

Claude Code Creates a Branch

A Claude Code session works on a solution and pushes it to a new branch in your repository.

02

GitHub Action Deploys

The push triggers a GitHub Action that deploys to Cloudflare Pages with a unique preview URL.

03

Review & Merge

Visit the preview URL, review the solution, and merge to main when you're happy. Production updates instantly.
